PacketMachine Developers Document
====================================

`PacketMachine` is a library of packet sniffing and decoding in C++.

- Fast decoding Ethernet and TCP/IP
    - Multi threading
    - Less data copy
- Extensibility, basic support protocols
    - Ethernet, PPPoE
    - ARP
    - IPv4
    - TCP, UDP, ICMP
    - DNS, mDNS
    - DHCP
- Based on C++14
