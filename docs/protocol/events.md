Events
===========

| Event name       | What is captured / happend      |
|:-----------------|:--------------------------------|
| `Ethernet`       | Ethernet packet                 |
| `ARP`            | ARP packet                      |
| `ARP.request`    | ARP request                     |
| `ARP.reply`      | ARP reply                       |
| `PPPoE`          | PPPoE packet                    |
| `IPv4`           | IPv4 packet                     |
| `ICMP`           | ICMP packet                     |
| `UDP`            | UDP packet                      |
| `TCP`            | TCP packet                      |
| `TCP.new_session`| Observed a new TCP session      |
| `TCP.established`| Completed TCP 3 way handshake   |
| `TCP.closed`     | a TCP session closed            |
| `DNS`            | DNS packet                      |
| `DNS.query`      | DNS query                       |
| `DNS.reply`      | DNS reply                       |
| `mDNS`           | mDNS packet                     |
| `mDNS.query`     | mDNS query                      |
| `mDNS.reply`     | mDNS reply                      |
| `DHCP`           | DHCP packet                     |
