PacketMachine Developers Document
====================================

`PacketMachine` is a library of packet sniffing and decoding in C++.

- Fast decoding Ethernet and TCP/IP
    - Multi threading
    - Less data copy
- High extensibility, basic support protocols
    - Ethernet
    - ARP
    - PPPoE
    - IPv4/v6
    - TCP/UDP/ICMP
    - DNS
    - mDNS
    - DHCP
- Based on C++14
